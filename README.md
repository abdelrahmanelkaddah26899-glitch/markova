# 📊 Markova - Sales Dashboard

## لوحة تحكم المبيعات

تطبيق احترافي لعرض ومراقبة بيانات المبيعات من ملفات Excel مع رسوم بيانية تفاعلية.

### المميزات ✨

- 📁 تحميل ملفات Excel (XLSX/XLS)
- 📈 رسوم بيانية تفاعلية (خطوط، أعمدة، دوائر نسبية)
- 💰 عرض مؤشرات الأداء الرئيسية (KPIs)
- 🌙 تصميم داكن احترافي
- 📱 واجهة مستجيبة للأجهزة المختلفة
- 🌍 دعم اللغة العربية والنصوص من اليمين للي سار (RTL)

### المتطلبات 📦

- Node.js (v14 أو أحدث)
- npm أو yarn

### التثبيت 🚀

```bash
# استنساخ المستودع
git clone https://github.com/abdelrahmanelkaddah26899-glitch/markova.git

# الدخول للمجلد
cd markova

# تثبيت المكتبات
npm install

# تشغيل التطبيق
npm run dev
```

### الاستخدام 💻

1. افتح التطبيق في المتصفح
2. اضغط على زر "تحميل ملف Excel"
3. اختر ملف Excel يحتوي على البيانات
4. سيتم معالجة البيانات وعرض الإحصائيات والرسوم البيانية

### صيغة ملف Excel المتوقعة 📋

يجب أن يحتوي ملف Excel على الأعمدة التالية:

| التاريخ | الفرع | إجمالي الفاتورة | الكاش | الفيزا |
|--------|-------|-----------------|-------|--------|
| 2024-01-01 | فرع القاهرة | 1000 | 600 | 400 |
| 2024-01-02 | فرع الإسكندرية | 1500 | 900 | 600 |

### المكتبات المستخدمة 📚

- **React** - مكتبة JavaScript لبناء واجهات المستخدم
- **Recharts** - مكتبة الرسوم البيانية
- **XLSX** - معالجة ملفات Excel
- **Tailwind CSS** - نظام التصميم
- **Vite** - أداة البناء السريعة
